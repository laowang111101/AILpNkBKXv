# HR企业员工管理系统 java177

## 项目概述

HR企业员工管理系统是一款面向企业的内部管理工具，旨在帮助企业高效管理员工信息、考勤、薪资以及系统运行监控等。

## 技术栈

- 后端：Spring Boot、Spring Security、Redis、Quartz、JWT
- 前端：HTML

### 系统管理
- 菜单管理
- 字典管理
- 参数设置
- 通知公告
- 日志管理

### 系统监控
- 在线用户
- 定时任务
- 数据监控
- 服务监控
- 缓存监控

### 系统工具
- 表单构建
- 代码生成
- 系统接口

### 组织架构
- 部门管理
- 岗位管理
- 角色管理
- 员工管理

### 员工假勤
- 考勤规则
- 考勤打卡
- 请假管理
- 我要请假
- 请假审批
- 加班管理

### 员工薪资
